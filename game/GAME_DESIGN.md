# Game Design: Project Silver-Shadow (Temporary)

## 1. 개요
- **장르:** 중세 판타지 덱 빌딩 로그라이크
- **플랫폼:** PC (Steam 패키지 판매 목표)
- **개발 환경:** Godot Engine (GDScript)

## 2. 핵심 시스템
- **다대다 전투:** 최대 5:5 규모의 팀 전투.
- **타임바 시스템 (Active Turn):** 유닛의 '속도'에 따라 행동 순서 결정.
- **자원 시스템:** 캐릭터 턴 도달 시 보유 자원 내에서 카드 사용.
- **방어력(Block):** 턴 종료 시 소멸하는 방어 시스템 (혹은 전략에 따라 유지).

## 3. 초기 데이터 설계 (Cards)
- **Strike (타격):** 코스트 1, 피해 6
- **Defend (수비):** 코스트 1, 방어 5
- **Bash (강타):** 코스트 2, 피해 10, 취약 부여
